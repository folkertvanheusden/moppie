Required packages:
 * libsdl-gfx1.2-dev
 * libsdl1.2-dev
 * libsndfile1-dev
 * libsdl-image1.2-dev

Building:
 make

Note: the program can not (easily be) terminated.
That's because all key presses are intercepted to prevent your kid to accidently stopping the program.
Enter "STOP!" (upper case!) to stop the program.
